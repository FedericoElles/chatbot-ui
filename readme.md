# Simple UI for a Chatbot

Setup with the WiWo Test Chatbot

## Installation

    npm install

## Development

    gulp
    
    
## Embedding into HTML

			<div class="ajaxify2" data-command="chatbot" 
				data-header="true"
				data-feedback="true"
				data-intro="Haben Sie spezielle Fragen zum Artikel?"
				data-smartquestions="Wie geht es Dir?, Was machst Du so?"
			></div>
